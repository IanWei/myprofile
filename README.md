## Project setup
```
npm install
```

### Run the app
```
npm run serve
1. Visit localhost:8080
2. Select the Upload button at the bottom. 
3. Upload a png image from your local computer.
4. Adjust the slider bar to add effects on the image
5. Enjoy it!
```
